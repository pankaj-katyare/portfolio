Circles.Life

Circles.Life is a mobile virtual network operator, which means it operates as a mobileservice provider by leasing network infrastructure from existing telecommunication companies. This allows Circles.Life to offer mobile services to its customers without owning the physical network infrastructure. The system is built as Microservices based service mesh architecture deployed on kubernetes powered by Istio. Services communicate with each other with mutual TLS andgrpc as transport. To manage third party api throttling used GCP Cloud Tasks (Task queue) with Cloud functions written in GoLang on the other end. For asynchronous interservice Message queue used GCP PubSub and for frontend used React.JS with recoil for state management & websocket for notifications. Tech Stack : GoLang, Microservices, GRPC, Angular Firebase auth, AWS Kubernetes Engine, PostgreSQL, Cloud function. React, newRelic, MongoDB, GCP.
